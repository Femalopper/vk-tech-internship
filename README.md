## Description

Demonstration of adaptive form for choosing negotiation room. 

## Setup

```sh
   git clone git@github.com:Femalopper/vk-tech-internship.git

   cd vk-tech-internship

   npm ci
```

## Run app

```sh
   cd vk-tech-internship

   npm start
```
